# Windows Server Active Directory Home Lab

## Project Overview

I created a virtual Active Directory environment using Windows Server and a Windows 11 client. I used this lab to practise domain administration and common IT support tasks, including creating users, resetting passwords, joining a computer to the domain, and configuring Group Policy settings.

## Technologies Used

- Oracle VirtualBox
- Windows Server 2022
- Windows 11
- Active Directory Domain Services
- Group Policy
- DNS
- DHCP


## Tasks Completed

- Created Windows Server 2022 and Windows 11 virtual machines in Oracle VirtualBox
- Configured Windows Server as a domain controller
- Installed and configured Active Directory Domain Services
- Joined the Windows 11 client to the domain
- Created and managed domain user accounts
- Practised resetting user passwords
- Configured a domain password policy through Group Policy
- Changed the Windows 11 desktop wallpaper through Group Policy
- Practised basic DNS and DHCP configuration
-  Created a shared folder on Windows Server
- Assigned folder access to a domain user
- Accessed the shared drive from the Windows 11 domain client
